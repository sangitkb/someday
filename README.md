# 🚀 SomeDay

A modern bucket list and goal tracking application built with React.

## ✨ Features

- ➕ Add Goals
- ✏️ Edit Goals
- 🗑️ Delete Goals
- ✅ Mark Goals as Completed
- 🔍 Search Goals
- ↕️ Sort Goals
- 📊 Live Statistics
- 💾 Local Storage Persistence
- 🎨 Responsive UI
- ✨ Framer Motion Animations

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- React Icons
- date-fns

## 📂 Project Structure

```
app/
├── src/
│   ├── components/
│   ├── pages/
│   ├── services/
│   ├── hooks/
│   ├── utils/
│   └── context/
```

## 🚀 Installation

```bash
git clone https://github.com/sangitkb/someday.git
cd someday/app
npm install
npm run dev
```
